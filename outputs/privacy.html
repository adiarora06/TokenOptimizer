Privacy Policy

Token Optimizer Privacy Policy

This policy explains how Token Optimizer and the Token Optimizer for Gemini Chrome extension handle prompt text and related data.

Effective date: July 16, 2026 Single purpose: prompt optimization No auto-send

Plain Summary

  • Prompt text is sent to Token Optimizer only when you click Optimize.
  • Nothing is sent when you capture text or insert text into Gemini.
  • The extension does not auto-send Gemini messages.
  • The extension does not store provider API keys, passwords, or account credentials.

Single Purpose

Token Optimizer for Gemini has one purpose: helping users clean up, shorten, and clarify prompt text before inserting the reviewed result into Gemini.

Data We Process

  • Prompt text: text you paste into Token Optimizer or ask the extension to capture from Gemini.
  • Optimized prompt output: the rewritten prompt returned by Token Optimizer so you can review, copy, or insert it.
  • Local browser data: the extension may store the latest raw prompt locally to avoid recursively optimizing an already optimized prompt.
  • Operational request data: basic technical information may be processed by hosting infrastructure to deliver, secure, and debug the service.

How We Use Data

  • To convert rough or repetitive prompt text into cleaner prompt text.
  • To return the optimized prompt to the user for review.
  • To support the extension workflow: capture, optimize, insert, and review.
  • To maintain security, reliability, and debugging for the Token Optimizer service.

Data Sharing

Prompt text may be processed by backend infrastructure and AI model-processing services solely to provide the prompt optimization feature. We do not sell user data. We do not use or transfer user data for advertising, creditworthiness, lending, or unrelated purposes.

What We Do Not Collect

  • We do not intentionally collect names, addresses, payment information, health information, passwords, or authentication credentials.
  • We do not collect browsing history.
  • We do not perform keystroke logging, mouse tracking, or general user activity monitoring.
  • We do not automatically read Gemini conversations. Capture happens only when the user requests it.

User Control

  • You choose when to paste or capture prompt text.
  • You choose when to click Optimize.
  • You choose whether to copy or insert the optimized prompt.
  • You choose when to send the message in Gemini. The extension does not send it for you.
  • You can remove locally stored extension data by clearing the extension's site data or removing the extension.

Retention

The extension stores the latest raw prompt locally in your browser only to prevent recursive prompt wrapping. Backend systems may process prompt text long enough to provide the optimization response and maintain service reliability. Token Optimizer does not use prompt text for advertising or resale.

Children

Token Optimizer is not directed to children under 13. Users should not submit sensitive personal information, secrets, credentials, or private data they do not want processed for prompt optimization.

Changes

This policy may be updated as Token Optimizer changes. The effective date at the top of the page will be updated when material changes are made.

Contact

For privacy questions, open an issue at github.com/adiarora06/TokenOptimizer/issues.
